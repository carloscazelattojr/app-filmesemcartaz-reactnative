# App Filmes Em Cartaz - ReactNative

App desenvolvido para fins de estudo. Conecta em uma API de lista de filmes em cartaz e apresenta foto e nome dos filmes.

Requisição feita via GET

## Tecnologias

`React-Native` `Javascript`
